# Admin Dashboard (CRA + Bootstrap)
This is a Create React App project scaffold with:
- Bootstrap for styling
- Modular components under `src/components`, `src/pages`, `src/layouts`, `src/context`
- Theme (color + dark mode) persisted to localStorage
- Lazy-loaded routes using `React.lazy` + `Suspense`

To run:
1. `npm install`
2. `npm start`
